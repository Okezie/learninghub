FUNDAMENTALS OF WEB DEVELOPMENT (HTML, PHP, MySQL)

Duration: 12 weeks (108 hours)

Lecture Days: 3 times/week (3 hours/day)

Certificate Awarded: Diploma in Web Development

Professional Certification: Zend PHP , MySQL Developer

Objectives:

At the end of the programme, participants would be able to:

  • Work with text, breaks, graphics, links, marquee and comments in HTML
  • Perform diverse operations on tables in HTML
  • Use forms, checkboxes, drop boxes, list boxes and radio buttons in HTML
  • Create and modify a PHP page
  • Working with variables and data types
  • Use if/elseif/else statements to control processing conditionally
  • Create programs that include for, while, and do loops to process statements repeatedly
  • Employ the break, continue, and exit statements to modify default loop behaviours
  • Create arrays
  • Manipulate strings in PHP using the built-in functions
  • Query and administer a MySQL Database

Course content:

  • Introduction to HTML
  • Performing basic operations in HTML
  • Working with forms in HTML
  • Overview of PHP
  • Basic scripting and looping construct
  • Conditional constructs
  • PHP functions
  • PHP operators
  • Arrays in PHP
  • Working with databases and forms
  • Working with datafiles in PHP
  • Introduction to the Apache web server
  • Enabling E-Commerce
  • Querying and Administering the MySQL database
