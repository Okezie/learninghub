FUNDAMENTALS OF JAVA PROGRAMMING

Duration: 8 weeks (72 hours)

Lecture Days: 3 times/week (3 hours/day)

Certificate Awarded: Diploma in Java Programming

Professional Certification: Oracle Certified Associate Java Programmer (OCAJP)

Objectives:

At the end of the programme, participants would be able to:

  • Understand the Java technology stack
  • Understand the Java programming language
  • Understand the Java product lifecycle
  • Understand object-oriented concepts in Java
  • Use the Java language to create Java applications

Course content:

  • Java technology primer
  • Developing and testing a Java program
  • Classes, Objects and Relationships
  • Programming with Java statements
  • Conditional and loop constructs
  • Create and use objects
  • Understanding methods
  • Encapsulations and Constructors
  • Strings, Wrapper classes and Casting objects
  • Collections
  • Understanding Polymorphism
  • Understanding Inheritance
  • Handling exceptions
